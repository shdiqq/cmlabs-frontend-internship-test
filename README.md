# cmlabs-frontend-internship-test

## How to Run the Program
Clone Repository  `git clone https://github.com/shdiqq/cmlabs-frontend-internship-test.git`

### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run build:dev
```

#### Compiles and minifies for production
```
npm run build:prod
```
